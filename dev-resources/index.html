Installing Atom packages on Windows behind a proxy

August 14, 2016

If you’d like to install Atom packages on Windows behind a proxy, you have to configure proxy settings in ~/.atom/.apm/.apmrc, and set the ATOM_NODE_URL environment variable.

 ~/.atom/.apm/.apmrc
     http-proxy=http://&lt;user&gt;:&lt;password&gt;@&lt;host&gt;:&lt;port&gt;  
     https-proxy=http://&lt;user&gt;:&lt;password&gt;@&lt;host&gt;:&lt;port&gt;
Path
    ATOM_NODE_URL= http://gh-contractor-zcbenz.s3.amazonaws.com/atom-shell/dist

Others say that you have to add strict-ssl=false to .apmrc, but it doesn’t work for me.
